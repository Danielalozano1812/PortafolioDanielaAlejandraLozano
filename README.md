# Portafolio Daniela Lozano · v3 Cherry Edition

Portafolio web personal — Comunicadora Social y Líder de Marketing Digital.

## Estructura del proyecto

```
portafolio-daniela-v3/
├── index.html          ← página principal
├── styles.css          ← estilos (sistema Cherry)
├── script.js           ← interacciones
└── img/                ← 23 imágenes optimizadas
```

## Sistema visual: Cherry Editorial

Inspirado en branding tipo Cherry Chewy con paleta rosa cereza + acentos románticos.

### Paleta de colores

- `--cherry: #65001E` — color principal (burgundy oscuro)
- `--cream: #FFE9EC` — fondo crema rosado
- `--pink: #FFBACF` — acentos y hover states
- `--rose: #B05D76` — rosa apagado para secciones secundarias
- `--charcoal: #2B2B2B` — texto principal

### Tipografías

- **Montserrat** (900) — titulares principales, navegación, UI
- **Dancing Script** (600) — acentos románticos, frases destacadas
- **Newsreader** — cuerpo del texto, copy editorial

## Características técnicas

- ✅ Responsive completo (desktop, tablet, mobile)
- ✅ Floating WhatsApp sutil (solo ícono, 54px)
- ✅ Foto de Daniela protagonista con anillo decorativo y 4 badges editoriales
- ✅ Animaciones de scroll reveal con IntersectionObserver
- ✅ Smooth scroll con offset para nav fija
- ✅ Lazy loading en imágenes (hero eager para LCP rápido)
- ✅ Favicon SVG inline (sin requests adicionales)
- ✅ theme-color para mobile browsers
- ✅ prefers-reduced-motion respetado
- ✅ Imágenes optimizadas (3.4MB → 2.7MB)

## Cómo subir a GitHub Pages

1. Entra a tu repo: `Danielalozano1812/portafoliodanielalozano`
2. Borra los archivos antiguos del repo
3. Sube los archivos nuevos:
   - `index.html`
   - `styles.css`
   - `script.js`
   - Carpeta `img/` completa
4. Commit message: `"Actualización v3 — Cherry Edition"`
5. Espera 2 minutos
6. Refresca https://danielalozano1812.github.io/portafoliodanielalozano/
